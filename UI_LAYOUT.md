## 画面構成

### Nav (左パネル、固定幅 220px): プレイヤー情報
- **プレイヤー情報**: 現在のステータス（HP、レベル、攻撃力、防御力、経験値、ゴールド）
- **現在地情報**: ダンジョンの階層と探索進捗、または拠点表示

### Workspace (中央パネル、可変幅): メイン操作域
モードに応じて以下のいずれかの表示が排他的に切り替わります:

- **メインメニュー** (MainMenu / GameOver 時): プレイヤー名入力、ゲーム開始ボタン
- **拠点** (InBase 時): HP回復プログレスバー、再遠征ボタン
- **戦闘** (InCombat 時): 敵情報表示
- **ダンジョン探索ボタン**: 探索、休息、ポーション使用

### Bottom (下部パネル、折りたたみ可): ゲームログ
- 最新15件のゲームイベントを表示
- スクロール可能
- 戦闘ログ、探索結果、レベルアップ通知など
- **折りたたみ可能**: トグルボタンでログ表示/非表示を切り替え

## ゲームフロー

1. **メインメニュー**: プレイヤー名を入力して「ゲーム開始」
2. **ダンジョン探索**: 「部屋を探索する」でランダムイベント
3. **戦闘**: 敵に遭遇したら「攻撃する」か「逃げる」を選択
4. **勝利**: 敵を倒すと経験値とゴールドを獲得
5. **ボス戦**: 5部屋ごとにボス出現、倒すと次の階へ
6. **Game Over**: HPが0になるとゲームオーバー、メインメニューに戻る
